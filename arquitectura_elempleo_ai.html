05 · Modelo de Costos Claude API
Haiku 4.5 · Clasificación y extracción

$0.25 input · $1.25 output por 1M tokens

Usado por: Churn Predictor, Matching Notifier, Employer Signal Agent. Ideal para tareas simples, rápidas y baratas.

Ejemplos: ~$0.0004/usuario (churn) · ~$0.001/alerta (employer)
Sonnet 4.6 · Generación y razonamiento

$3.00 input · $15.00 output por 1M tokens

Usado por: Job Match Agent, Early Activation Agent, Re-engagement Agent. Para mensajes personalizados de alta calidad.

Ejemplos: ~$0.023/búsqueda (match) · ~$0.010/mensaje (reengagement)
